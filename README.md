# Dev-lab
Ambiente de testes e desenvolvimento para práticas em Python, Javascript e fundamentos de software.

## FocusFlow
Este repositório contém um protótipo funcional do app FocusFlow (Web, Android e iOS) criado com Expo + React Native.

### Executar localmente
1. Instale dependências:
   ```bash
   npm install
   ```
2. Inicie o projeto:
   ```bash
   npm run start
   ```
3. Abra no destino desejado:
   - Web: pressione `w`
   - Android/iOS: use um emulador ou o app Expo Go
